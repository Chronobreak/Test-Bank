Poker
=========

Prompt
======

Create a function that takes in two poker hands and returns the winning hand.

Each hand will be an array of five objects, where each object has a suit and value property.

Example to give
```
Input: [4, 1, 8, 7]
Output: True
Explanation: (8-4) * (7-1) = 24

Input: [1, 2, 1, 2]
Output: False

```
