# Two Sums

## Prompt

Write a function that when given an array of numbers and a target integer will return any two integers within the array that could add up to the target value.

Your function must run in linear time!

Examples to give

```
Inputs: [1,3,5] , 6
Output: [1,5]

Inputs: [1,2,3,4] , 7
Output: [3,4]

```
